# Profiler task

## Build & Run:
* Build:
    ```bash
    make
    ```
* Run:
    ```bash
    ./bin/main <file_name>
    ```
    ```bash
    ./bin/main_fork <file_name_1> <file_name_2> ...
    ```
    ```bash
    ./bin/etalone <file_name> # just as main, but with c funcs instead of syscalls
    ```

## Test:
```bash
python3 test.py
```
